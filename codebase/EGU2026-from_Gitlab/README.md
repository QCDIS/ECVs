# Analytical Workflow Templates
## Overview
This repository contains a curated collection of [Common Workflow Langage](https://www.commonwl.org/) templates meant to showcase how the services and resources provided in the [ENVRI-Hub](https://envri-hub.envri.eu/) can be used to create meaningful data processing flows.
